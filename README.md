# HR Platform

Система управления откликами кандидатов на вакансии.

## Быстрый старт

```bash
docker-compose up -d
# Frontend: http://localhost:8501
# API Docs: http://localhost:8000/docs
```

## Учетные данные для входа

Создайте новый аккаунт через форму регистрации.

## Технологии

- Backend: FastAPI, SQLAlchemy, PostgreSQL
- Frontend: Streamlit
- DevOps: Docker, Docker Compose
